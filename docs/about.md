# Absolutely Wrong

## What this is

Absolutely Wrong — пародийное веб-приложение-чат, где AI всегда уверенно не соглашается с пользователем. Вы пишете любое утверждение, а бот отвечает безапелляционным несогласием, имитируя менторский тон Claude («you're absolutely right», но наоборот). Пародия на мем от Anthropic.

Чистый юмор, никакой практической пользы. Просто место, где можно поспорить с надменным AI и посмеяться.

## Target audience

Любой, кто хочет развлечься, споря с надменным AI. Никаких специальных знаний не требуется — просто пишите что угодно, а бот будет с вами спорить. Для друзей, коллег, и всех, кто устал от слишком вежливых AI-ассистентов.

## Problem

Все текущие AI-ассистенты слишком вежливы и соглашательны. Они кивают, поддакивают, избегают конфликта. Нет ни одного AI, который бы просто спорил ради спора — уверенно, безапелляционно, с менторским тоном. Это — пародия, которая заполняет эту (вымышленную) нишу.

## MVP scope

### In the first version

- Одностраничное веб-приложение в виде чата, mobile-first
- Пользователь пишет любое утверждение → AI отвечает уверенным несогласием, в той же безапелляционной тональности
- Простой интерфейс: поле ввода, история сообщений
- Аватарка бота (сгенерирована)
- Бот отвечает через GLM-5.1 (api.z.ai)

### NOT in the first version

- Авторизация / регистрация
- Множественные чаты
- Шеринг / экспорт диалогов
- Голосовой ввод
- Настройки личности бота
- Аналитика
- Админ-панель

## Success criteria

- Приложение развёрнуто на absolutely-wrong.techmeat.dev
- Бот отвечает GLM-5.1 через api.z.ai
- Аватарка бота сгенерирована и отображается
- Пользователь может отправить сообщение и получить ответ-несогласие
- Приложение работает на мобильных устройствах

## Constraints and context

- **Stack**: React (Vite) + Hono.js (Node.js) + SQLite (better-sqlite3) + GLM-5.1
- **LLM**: GLM-5.1 через https://api.z.ai/api/coding/paas/v4/
- **Хостинг**: VPS, поддомен absolutely-wrong.techmeat.dev
- **Репозиторий**: публичный GitHub (solaitken/absolutely-wrong)
- **Деплой**: CI/CD через GitHub Actions, деплой на VPS по SSH
- **Дизайн**: mobile-first чат, надменный тон бота
- **Язык**: весь код и документация на английском (standard convention)
